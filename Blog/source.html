The Fourier Awakens

Robbie Katz (KTZROB006) | Benji Lewis (LWSBEN002) | Cairin Michie (MCHCAI001)

Project Overview

In this project a direct digital synthesizer1 will be designed. This digital synthesizer will be used to implement a Fourier Series estimation. The user specifies the waveform, as well as the amplitude, phase and frequency of the waveform that they want synthesized on the FPGA. The user can also specify the number of terms in the Fourier estimation. The FPGA processes this information with the use of the Fourier series equation, and outputs the results.

Prototype Specification

  • A CLI will be developed for the user to specify the waveform, and desired parameters.
  • The information required by the FPGA for the waveform synthesis will be passed to the FPGA via a text file. This will serve as a look-up table for the DDS modules.
  • The DDS modules each output a sine wave, with amplitude, phase and frequency as specified by the look-up table. The phase is used to convert the sine wave to a cosine wave2. This is done because the Fourier Series equation requires a summation of sine and cosine waveforms.
  • The output of each DDS module is summed, to produce the final waveform.
  • The final waveform will be outputted through a DAC. If time permits, an oscilloscope module will be developed that will be used to display the output waveform on a monitor using the VGA connection.
